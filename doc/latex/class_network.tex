\doxysubsection{Detailed Description}
A neuron network is a set of neurons and a set of directional links between them.

Neurons are objects of class \mbox{\hyperlink{class_neuron}{Neuron}} (they are identified by their index in the vector neurons). A link is an ordered pair of indices in neurons with an intensity value, collected in a linkmap. This is a \href{https://en.cppreference.com/w/cpp/container/map}{\texttt{ std\+::map}} therefore only one connection can exist between two neurons.

To create a network, you need to \mbox{\hyperlink{class_network_ae9d6a1913ae92e1d2d7cd17f70f815b5}{resize}} it and optionally \mbox{\hyperlink{class_network_ad1d20020028425cfab199da1942172c9}{set\+\_\+default\+\_\+params}} for each neuron. Then you can either call \mbox{\hyperlink{class_network_a6ebe0899329973e4924997a25e205856}{add\+\_\+link}} for each connection or generate a random network with \mbox{\hyperlink{class_network_a4c751a78c7bc5e27c6b36df78c9c70c0}{random\+\_\+connect}}.

The dynamics of the network proceeds by calling \mbox{\hyperlink{class_network_a53665a3a69e6ec894c313d0ce3fb7f34}{step}}. The state of the network can be printed to output streams with print\+\_\+params (to print all parameters of all neurons), print\+\_\+traj to print the full state of one neuron of each type. The helper function print\+\_\+head will print a header line with the variable names for the the print\+\_\+traj lines.

This class provides accessors to the following \mbox{\hyperlink{class_neuron}{Neuron}} properties
\begin{DoxyItemize}
\item \mbox{\hyperlink{class_network_a084774282bb3c1129a2931e3277ddb90}{degree}} \+: returns the degree of a neuron (number of incoming connections) and valence (sum of weights of these links)
\item \mbox{\hyperlink{class_network_a98bbf44b077ba1bec5a62ecffc405503}{neighbors}} \+: returns the indices of neurons with incoming links to a given neuron,
\item potentials \+: returns the values of membrane potentials for all neurons,
\item recoveries \+: returns the values of recovery variables for all neurons,
\item neuron \+: returns a const reference to a given neuron object. 
\end{DoxyItemize}
